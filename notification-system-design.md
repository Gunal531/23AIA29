# Notification App Frontend

A simple React-based notification system frontend for real-time notifications.

## What it does
Displays notification cards in a clean interface, fetching updates from the backend API.

## Tech Stack
- React 18
- Vite
- JavaScript/JSX
- ESLint for code quality

## Installation
```bash
npm install
```

## Running the app
```bash
npm start
```

The app will run on `http://localhost:5173`

## Project Structure
- `src/components/` - React components
- `src/api/` - API calls to backend
- `src/App.jsx` - Main application component
